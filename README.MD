# WordPress サポートフォーラム ユーザーガイド

WordPress.org 日本語サポートフォーラムで、より効果的に質問・回答するためのガイド集です。

## 📚 このプロジェクトについて

このリポジトリは、WordCamp Kansai 2025 コントリビューターデイ Support テーブルの活動から生まれました。  
WordPress コミュニティのメンバーが協力して、フォーラムを使う全ての人のためのガイドを作成しています。

## 🎯 目的

- **質問者**が効果的に問題を解決できるようサポート
- **回答者**が安全で的確な回答を提供できるようサポート
- **フォーラム全体**のコミュニケーション品質向上

## 📖 ガイド一覧

### 質問者向けガイド
質問者が効果的に問題を解決できるようサポートするガイドです。

- [フォーラム質問ガイド](how-to-question/how-to-question.md) - 効果的な質問をするためのポイント

### 回答者向けガイド
回答者が安全で的確な回答を提供できるようサポートするガイドです。

- [フォーラム回答ガイド](how-to-answer/how-to-answer.md) - 安全で的確な回答をするためのポイント（作成中）

### イベント資料
- [WordCamp Kansai 2025 コントリビューターデイ Support テーブル活動案](wckansai2025-contributor-day/contributor-day-support-table.md)

## 🤝 コントリビューション

このガイドは WordPress コミュニティ全体の財産です。  
改善提案や追加コンテンツは大歓迎です！

### 貢献方法

1. このリポジトリをフォーク
2. 新しいブランチを作成 (`git checkout -b feature/improvement`)
3. 変更をコミット (`git commit -am 'ガイドを改善'`)
4. ブランチにプッシュ (`git push origin feature/improvement`)
5. プルリクエストを作成
